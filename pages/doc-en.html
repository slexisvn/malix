By default, operations are performed in RADIAN

Calculate

  • R Radian
  • D Degree
  • G Gradian

Functions

  • approxratio(a, decimal) Returns the repeating decimal. Eg. approxratio(2,333) -> 7/3, approxratio(2,21333) -> 166/75
  • abs(x) Returns the absolute value of a real number or the magnitude of a complex number
  • sqrt(x) Returns the square root of x
  • mod(a, b) Returns the remainder of the result of a divided by b
  • pfactor(x) Returns the prime factors of x
  • exp(x) Returns the exponential of x
  • min(a, b, ...) Returns the min of a set of numbers
  • max(a, b, ...) Returns the max of a set of numbers
  • floor(x) Returns the largest integer not greater than x
  • ceil(x) Returns the smallest integer not less than x
  • fact(x) Can be entered as x!
  • dfactorial(x) Can be entered as x!!
  • log(a, b) Returns the logarithm of a base b
  • ln(a) Returns the natural logarithm of x
  • log10(x) Returns the logarithm of a base 10
  • nCr(n, k) Returns the number of combinations of n items taken k at a time
  • nPr(n, k) Returns the number of permutations of n items taken k at a time
  • round(x) Rounds x to the nearest integer
  • round(x, n) Round x to n significant digits
  • divisors(x) Returns the divisor of an integer x
  • sign(x) Returns the sign of x
  • Trigonometric: cos, sin, tan, sec, csc, cot, acos, asin, atan, asec, acsc, acot
  • Hyperbolic trig: cosh, sinh, tanh, sech, csch, coth, acosh, asinh, atanh, asech, acsch, acoth
  • arg(z) Returns the angle of complex z
  • imagpart(z) Returns the imaginary part of complex z
  • realpart(z) Returns the real part of complex z
  • conjugate(z) Returns the complex conjugate of z
  • polarform(z) Returns complex z in polar form
  • rectform(z) Returns complex z in rectangular form
  • Si(x), Ci(x), Ei(x), Shi(x), Chi(x) Returns the trigonometric integral
  • rect(x) Returns the rectangular function
  • tri(x) Returns the triangular function
  • step(x) Returns the Heaviside step function
  • sinc(x) Returns the cardinal sine function
  • erf(x) Returns the error function of x
  • gamma(x) Returns the Gamma function of x
  • beta(a, b) Returns the Beta function of a, b
  • zeta(x) Returns the Zeta function of x
  • bernoulliN(n) Returns the nth Bernoulli number
  • eulerN(n) Returns the nth Euler number
  • bell(n) Returns the nth Bell number
  • stirling1(n, k) Returns the Stirling numbers of the first kind
  • stirling2(n, k) Returns the Stirling numbers of the second kind
  • catalan(n) Returns the nth Catalan number

Solve

Support

  • Solve polynomial equations, simple trigonometry equations, simple logarithmic equations
  • Solve polynomial inequalities of degree 2, 3, 4
  • Solve rational inequalities

Polynomial

Mode

  • E Expands a function or expression. Functions
    • hermite(n) Returns the nth "physicists" Hermite polynomial
    • hermiteE(n) Returns the nth "probabilists" Hermite polynomial
    • legendreP(n) Returns the nth Legendre polynomial
    • laguerre(n) Returns the nth Laguerre polynomial
    • lagrange Returns the Lagrange polynomial
    • bernoulliP(n) Returns the nth Bernoulli polynomial
    • chebyshevT(n) Returns the nth Chebyshev polynomial of the first kind
    • chebyshevQ(n) Returns the nth Chebyshev polynomial of the second kind
    • eulerP(n) Returns the nth Euler polynomial
  • F Factor an expression
  • C Finds the coefficient of x to the n
  • P Performs partial fraction decomposition

Calculus

Mode

  • L Limit
  • D Derivate
  • I Integral
  • T Definite integral
  • E Taylor expansion

In mode D, if x0 is omitted, the derivative at point isn't calculate

In mode E, if x0 is omitted then x0 = 0 is used

Matrix

Functions

  • determinant(m) Returns the determinant of matrix m
  • invert(m) Returns the inverse of matrix m
  • imatrix(n) Returns an n by n identity matrix
  • transpose(m) Transposes a matrix
  • adjunct(m) Returns the adjunct of matrix m
  • hilbert(n) Returns an n by n Hilbert matrix

Vector

Functions

  • abs(u) Returns the vector length
  • dot(u, v) Returns the dot product of vectors u and v
  • cross(u, v) Returns the cross product of vectors u and v
  • angle(u, v) Returns the angle of vectors u and v
  • pointM(A, B) Returns the midpoint coordinates of point A and point B
  • pointG(A, B, C) Returns the centroid coordinates of triangle ABC
  • pointH(A, B, C) Returns the orthocenter coordinates of triangle ABC
  • pointI(A, B, C) Returns the incenter coordinates of triangle ABC
  • pointO(A, B, C) Returns the circumcenter coordinates of triangle ABC
  • line(A, B) Returns the equation of a line through point A and point B in the plane
  • plane(A, B, C) Returns the equation of a plane through point A, point B and point C in space
  • projL(A, u, M) Returns the projection of point A onto the line (x - x0) / a = (y - y0) / b = (z - z0) / c, with u = (a; b; c) is the directon vector and point M0(x0; y0; z0) is belong to this line
  • projP(A, n, D) Returns the projection of point A onto the plane Ax + By + Cz + D = 0, with n = (A; B; C) is the normal vector of this plane

Number sequence

Mode

  • U Finds the general term of number sequence
  • S Finds the general sum of number sequence
  • O Functions
    • fib(n) Returns the nth Fibnoacci number
    • lucas(n) Returns the nth Lucas number
    • farey(n) Returns the nth Farey sequence
